The latest stable releases of Firefox, Chrome and Opera are supported. Midori is a bad browser and you should feel bad for using it.

Chrome: Install 4chan X, that's it.

Firefox:

  • If you don't have either Scriptish or Greasemonkey installed, install Scriptish (it can auto-update scripts), restart Firefox, and then install 4chan X.
  • to disable compatibility checking: go to about:config > right-click > new > boolean > extensions.checkCompatibility.nightly > false. Or you can get the Nightly Tester Tools.

Opera: Save 4chan X into your userscripts folder. If you haven't already, select a folder with: Ctrl + F12 (Preferences) > Advanced > Content > Javacript Options > User Javascript Folder.

Bug reports: before making one,

  1. Update your browser / 4chan X to the latest stable.
  2. Disable your other extensions / scripts.
  3. If your issue persists, report console errors, if any. Open your console with: Ctrl + Shift + J on Firefox / Chrome, Ctrl + Shift + I on Opera.

Clearing your 4chan cookies will erase your settings!

Firefox: If you get an error stating localStorage is null, go to about:config and enable dom.storage.enabled.

Chrome:

  • Some userstyles, notably AppChan, break text input due to a WebKit bug. [89647]
  • 4chan X only runs on 4chan; Chrome will say that it runs on all domains due to its match patterns.

*nix: download all images in a thread with this shell function (add it to your .bashrc):

dta() {
  wget -O - $1 |
  grep -Eo 'http://images.4chan.org/[^"]+' |
  uniq |
  xargs wget
}
