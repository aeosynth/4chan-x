Installation Instructions:(Remember to refresh 4chan [after installing] to see changes.)

Chrome:

Click the button at the top of the page. That's it.

Firefox:

If you haven't already, install either Scriptish or Greasemonkey.(See the FAQ for comparison.)

Restart Firefox, come back to this page and click the button at the top of the page to install.

Opera:

Save 4chan X into your userscripts folder. If you haven't, select a folder with CTRL + F12 (Preferences) > Advanced > Content > JavaScript Options > User JavaScript Folder.

Bug Reports:

Before making one:

  1. Update your browser & 4chan X to the latest stable versions.
  2. Disable your other extensions, scripts or userstyles and try again.
  3. If your issue persists, report console errors (if any). Open your console with CTRL + SHIFT + J on Chrome/Firefox or CTRL + SHIFT + I if you're on Opera.

Firefox Notes:

It should be noted that Firefox 3.5 is not supported.

What's the difference between Scriptish and Greasemonkey?

Scriptish auto updates scripts for you. That's all you really need to know. If you need to learn more, they invented this thing called Google, try using it.

But Scriptish/Greasemonkey isn't supported on my version of Nightly or Aurora, since Firefox 6 is too mainstream! How do I make it work?

You have to disable compatability checking. You can either install the Nightly Tester Tools extension, or do it manually by about:config, etc...
